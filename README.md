# JSON Parser – Training Project
A small educational project implementing a basic JSON parser in C++.

# Requirements
- C++17 compatible compiler (e.g., MSVC)
- CMake (≥ 3.14)
- GoogleTest (for unit tests)

# Setup
Install dependencies:
Make sure you have a C++17 compiler and CMake installed.
GoogleTest can be either:
Automatically downloaded during CMake configuration, or
Manually placed into the extern/ directory as source code.
